Lead Automation System

This was a project I started back in June 2020, Brightstar Holdings approached me about creating a system where prospective clients could input data through a form and have it directly sent to a database. I was the sole creator/developer for the project, with help from the web development community on Github, Stack Overflow, etc.

The system stored customer data for the entire organization and provided an interface for project managers and admins to view, add and send data to clients.

Results would be generated by the system and automatically sent to the company directly from their website. The programming languages I used for this project were PHP, Javascript, HTML and CSS. As well as using back-end technologies such as AWS and MySql for data management.

The client had the ability to view all their data with a dynamic search form, trend data, and add notes.

I worked on this system for 3 months before licensing the code to Brightstar Holdings. I'm currently working on expaning licensing the code for other companies.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - React JS
  • - Postgres
  • - AWS (S3 & RDS)
  • - Zingcharts - Trending & graphing test results
  • - Google Maps API
  • - XML2PDF - PDF generator
